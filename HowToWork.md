# How to code section for the project

### example

index.html
```
<div id="sectionName">
    <div id="content">
        <h1>Sample Section Title</h1>
        <p>Description</p>
    </div>
    <div id="sectionNamePhone">
        <h1>Title on the phone</h1>
        <p>Paragraph on the phone</p>
    </div>
</div>
```

style.css
```
#sectionName #content h1 {
    color: white;
}

#sectionName #sectionNamePhone h1 {
    color: orange;
}
```
